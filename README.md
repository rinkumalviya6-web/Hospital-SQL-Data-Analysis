# Hospital-SQL-Data-Analysis
SQL project focused on analyzing hospital data to extract insights on patients, departments, expenses, and hospital performance.

This project focuses on analyzing hospital data using SQL to extract meaningful insights related to patients, departments, expenses, and
hospital performance.

## Tools Used
- SQL (PostgreSQL)

## Analysis Covered

 - Overall Patient Analysis
   
      Calculated the total number of patients treated across all hospitals to understand overall patient volume.
    
- Hospital-wise Doctor Availability
  
     Analyzed the average number of doctors available per hospital to evaluate resource distribution.
  
- Department-wise Patient Load
  
     Identified the top hospital departments with the highest number of patients to understand demand patterns.
  
- Medical Expenses Analysis
  
    Identified the hospital with the highest medical expenses
  
    Calculated daily average medical expenses per hospital
  
    Generated month-wise total medical expenses to analyze cost trends over time
  
- Length of Stay Analysis
  
    Calculated patient stay duration using admission and discharge dates
  
    Analyzed the average length of stay across departments
  
    Identified the longest hospital stay record
  
- Location-based Insights
  
    Analyzed city-wise total patients treated to understand geographical distribution of healthcare demand.

- Department Performance Evaluation
  
    Identified the department with the lowest number of patients for performance comparison.
